# パンダベビーカステラ 出店のご提案
**作成日：2026年6月21日**

---

## 1. 商品紹介

**パンダベビーカステラ**は、栃木県那須塩原市発のキッチンカー専門ベビーカステラです。  
パンダ型の愛らしいフォルムと焼きたてのふわっとした食感が特徴で、子どもから大人まで幅広い世代から支持されています。  
SNS映えする見た目から、お客様自身が写真を撮って拡散してくださることも多く、イベント会場の集客・話題づくりに大きく貢献します。  
食品営業許可を取得済みのキッチンカーによる現地製造販売のため、常に焼きたてをお届けできます。

---

## 2. 出店のメリット（施設・主催者の皆様へ）

- **集客力アップ**：パンダ型の見た目でお子様連れファミリーが立ち止まり、会場全体の滞在時間・回遊を促進します
- **SNS拡散効果**：お客様がSNSに投稿→施設・イベントの認知が自然に広がります（広告費ゼロ）
- **手間・コストなし**：当社が車両・調理設備・材料をすべて持参します。電源（100V 15A）のご提供だけで運営可能です
- **地域PRへの貢献**：栃木県那須塩原市発のローカルブランドとして「地域密着」「地産地消」のイメージに合致します
- **フレキシブル対応**：1日から対応可能。定期出店・単発イベントどちらもご相談ください

## 3. 出店条件・スペック

| 項目 | 内容 |
|------|------|
| 必要スペース | 約 3m × 6m（キッチンカー1台分） |
| 電源 | 100V・15A以上（延長コード持参可） |
| 水道 | あれば理想。なければ給水タンク持参で対応可 |
| 出店費用 | 売上歩合制（売上の○%）または固定出店料（要相談） |
| 標準営業時間 | 10:00〜17:00（応相談） |
| 人員 | 2名体制で運営 |
| 許認可 | 食品営業許可取得済み・移動販売届出済み |

## 4. 出店実績・信頼性

- 栃木県那須塩原市に本社を置く**株式会社芹江コンチェルト**が運営
- 旅館業（Lodgers Bldg SERIE）・フィットネスジム（Honey LaRva・大田原市＆那須塩原市）も展開する地域密着型企業
- 食品営業許可・移動販売届出をすべて法令に基づき取得済み
- キッチンカー・物産展の両形態での販売実績あり
